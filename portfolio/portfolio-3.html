Park Island Hub (project)

HTML, CSS, Javascript/jQuery, Jade, Express.js, Node.js, mongoDB, MVC architecture

This was a solo project working with node.js for the first time. The idea was to build a platform for managed residential estates in Hong Kong where residents could easily access up-to-date information, book facilities, check transport info etc.

Visit demo page (It may take a few moments to load)

  • Sign up/login via email or Facebook
  • Three user types - admin, resident, guest
  • Timetables of different transport routes with countdown
  • Admins can create new blog posts
