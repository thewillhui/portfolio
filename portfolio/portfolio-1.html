SimplyHome

Angular, Ionic, Ruby on Rails with Action Cable, Postgresql, MVC architecture

We were tasked to build a mobile app for a new startup in Hong Kong to help potential renters find user-rated real-estate agents. We had a 2 week timeframe and had to implement the following features:

  • Sign up/login for user and agent
  • Separate functionality for user and agent
  • User can send an enquiry
  • Agent is auto-matched to enquiry
  • Real-time chat system based on enquiry
  • Send listings and appointments via chat
  • User and agent rating system
